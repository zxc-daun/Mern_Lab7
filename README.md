# JWT_Authentication-_WithNodemailer

## Environment config
```
MONGO_URI = <YOUR_MONGO_URI>

JWT_SECRET_KEY = <YOUR_SECRET_KEY>

SMTP_EMAIL=<YOUR_GMAIL_EMAIL>

SMTP_PASS=<YOUR_GMAIL_PASSWORD>
```
